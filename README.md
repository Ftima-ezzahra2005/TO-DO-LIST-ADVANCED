# To Do List Advanced

## Description
Application avancée de gestion des tâches avec des fonctionnalités interactives en JavaScript.

## Technologies utilisées
- HTML
- CSS
- JavaScript

## Fonctionnalités
- Ajouter /close/show closed/Supprimer des tâches
- Sauvegarde des données
- Interface dynamique

## Aperçu

### Description:

Interface principale qui affiche toutes les tâches avec possibilité d’ajout, et close et supression.
